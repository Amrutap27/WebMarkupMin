   DESCRIPTION

   interfaces for all ASP.NET extensions.

   =============
   RELEASE NOTES
   =============
   1. Added support of ASP.NET 5 beta 6;
   2. Now during HTTP compression the deflate algorithm has a higher priority than
      the gzip algorithm.
