   ===========
   DESCRIPTION
   ===========
   WebMarkupMin.AspNet4.HttpModules contains 4 HTTP-modules:
   `HtmlMinificationModule` (for minification of HTML code),
   `XhtmlMinificationModule` (for minification of XHTML code),
   `XmlMinificationModule` (for minification of XML code) and
   `HttpCompressionModule` (for compression of text content by using GZIP or
   Deflate).
